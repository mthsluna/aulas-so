# Linha do Tempo: Evolução dos Sistemas Operacionais

Este documento apresenta a cronologia dos principais lançamentos de Sistemas Operacionais, conforme desenvolvido na atividade colaborativa via Miro.

---

## Cronologia Histórica

| Ano | Sistema Operacional | Marco Principal |
|:---:|:---|:---|
| **1969** | Unix | Introdução de conceitos de multitarefa e portabilidade. |
| **1981** | MS-DOS | Sistema operacional padrão para os primeiros computadores IBM PC. |
| **1984** | macOS | Lançamento da primeira interface gráfica (GUI) de sucesso comercial. |
| **1985** | Windows 1.0 | Primeira interface gráfica da Microsoft operando sobre o MS-DOS. |
| **1991** | Linux | Lançamento do Kernel por Linus Torvalds, marco do Software Livre. |
| **2001** | Windows XP | Unificação de robustez e facilidade de uso em uma nova arquitetura. |
| **2007** | iOS | Início da era dos smartphones modernos com foco em toque. |
| **2008** | Android | Sistema de código aberto que democratizou o acesso à tecnologia móvel. |
| **2009** | Windows 7 | Um dos mais populares, estável, rápido e amado até hoje. |
| **2012** | Windows 8 | Tentou ser tablet e pc ao mesmo tempo retirando o menu iniciar. |
| **2015** | Windows 10 | Trouxe o menu iniciar de volta e unificou o sistema em todos os dispositivos. |
| **2021** | Windows 11 | Design moderno com janelas arredondadas e icones centralizados na barra de tarefas. |
