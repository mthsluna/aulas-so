# Pesquisa: Sistemas Operacionais Baseados em Outros Sistemas

## 1. Introdução
Muitos sistemas operacionais modernos foram desenvolvidos a partir de outros já existentes, aproveitando seu kernel, arquitetura ou estrutura. Isso permite maior compatibilidade, segurança e redução no tempo de desenvolvimento.  

Este trabalho apresenta **5 sistemas operacionais populares**, suas origens, características e diferenças em relação ao sistema que serviu de base.

## 3. Descrição Detalhada dos Sistemas

### Ubuntu <img src="./assets/ubuntu.png" width="30" style="vertical-align: middle; margin-right:6px;">  

- **Origem:** Debian  
- **Foco:** Usuários domésticos e servidores  
- **Interface:** GNOME Desktop (tradicional, moderna e intuitiva)  
- **Recursos:** Atualizações regulares, compatibilidade de hardware ampla, repositórios oficiais com milhares de pacotes  
- **Mercado:** Computadores pessoais, servidores corporativos, cloud computing  

---

### Linux Mint <img src="./assets/linux-mint.png" width="30" style="vertical-align: middle; margin-right:6px;">  

- **Origem:** Ubuntu  
- **Foco:** Usuários que buscam familiaridade com Windows  
- **Interface:** Cinnamon, MATE ou Xfce  
- **Recursos:** Software pré-instalado, codecs multimídia incluídos, atualização simplificada  
- **Mercado:** Desktop doméstico, educação, novos usuários Linux  

---

### Android <img src="./assets/android.png" width="30" style="vertical-align: middle; margin-right:6px;">  

- **Origem:** Kernel Linux  
- **Foco:** Dispositivos móveis (smartphones, tablets, wearables)  
- **Interface:** Touchscreen, personalizável via launchers  
- **Recursos:** Google Play Store, multitarefa, apps nativos e de terceiros, integração com Google Services  
- **Mercado:** Smartphones, tablets, smart TVs, relógios inteligentes  

---

### macOS <img src="./assets/macos.png" width="30" style="vertical-align: middle; margin-right:6px;"> 

- **Origem:** Unix / NeXTSTEP  
- **Foco:** Computadores Apple (MacBook, iMac, Mac mini)  
- **Interface:** Aqua GUI, Dock, Mission Control  
- **Recursos:** Integração com iCloud, softwares nativos (Safari, Mail, GarageBand), segurança e estabilidade  
- **Mercado:** Profissionais criativos, usuários corporativos, consumidores premium  

---

### Chrome OS <img src="./assets/chromeos.png" width="30" style="vertical-align: middle; margin-right:6px;">  

- **Origem:** Linux / Gentoo  
- **Foco:** Computadores leves, educação e internet-first  
- **Interface:** Minimalista, baseada no navegador Chrome  
- **Recursos:** Aplicações web e Android, atualizações automáticas, segurança baseada em sandboxing  
- **Mercado:** Chromebooks, escolas, usuários que utilizam principalmente web apps
